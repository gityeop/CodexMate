# CodexMate 0.0.14

- Kept Desktop-started turns in the running state until a completion hint arrives, so quiet long-running commands no longer disappear from CodexMate after the short running-hint window.
- Added a safety timeout for inferred active turns when no completion signal is available.
- Prevented stale desktop running evidence from reviving threads that already have authoritative terminal activity.
- Added regression coverage for long-running quiet turns, pending-state preservation, completion handling, and stale running cleanup.
