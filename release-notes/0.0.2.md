# CodexMate 0.0.2

- Improved recent thread menu row sizing so titles align consistently, reserve icon space, and stay vertically centered.
- Hardened updater availability checks so builds without Sparkle metadata stay unavailable and invalid feed URLs surface a configuration issue.
- Updated packaging defaults to derive the GitHub Releases appcast URL from `origin` and load the Sparkle public key from the configured keychain account when available.
- Added regression coverage for thread row layout and updater configuration handling.
