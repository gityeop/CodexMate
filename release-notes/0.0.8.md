# CodexMate 0.0.8

- Reduced startup latency by showing a smaller recent-thread snapshot first and backfilling the full tracked list immediately after bootstrap.
- Made new thread discovery feel faster by boosting refresh cadence briefly when the app starts, returns to the foreground, or opens the menu.
- Avoided duplicate thread-list refreshes during bootstrap so the menu and notch no longer stall behind overlapping loads.
- Added a loading placeholder for recent threads and reduced notch overlay rebuild work while the menu is closed.
- Updated settings language refresh handling so localized titles change immediately, and restored the standard Command-W shortcut to close the settings window.
