# CodexMate 0.0.12

- Fixed a high-CPU app-server read loop when large JSON responses arrive in multiple stdout chunks.
- Reduced CPU spikes during live Codex activity by coalescing notification-driven menu renders and ignoring high-frequency token usage updates.
- Moved notch status animation playback to layer-backed frame animation so status sprites animate without a repeating Swift redraw timer.
- Added request timeouts for app-server RPC startup so the menu no longer stays stuck loading recent threads when initialize or thread list calls hang.
- Added regression coverage for app-server timeouts, noisy thread notifications, and notch sprite animation updates.
