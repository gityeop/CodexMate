# CodexMate 0.0.17

- Clear stale running indicators from Codex Desktop session files when a session records `task_complete` or `turn_aborted`.
- Suppress older running hints when a newer session completion is available, preventing completed threads from reappearing as running.
- Preserve session completion hints during database fallback so locked or temporarily unavailable state databases do not leave completed sessions stuck as running.
- Added regression coverage for completed sessions, newer completion hints, and fallback completion handling.
