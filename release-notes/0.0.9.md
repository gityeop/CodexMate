# CodexMate 0.0.9

- Hardened authoritative thread synchronization so archived or removed desktop threads drop out of the menu more reliably without regressing active or pending threads.
- Reduced desktop activity parsing, log scanning, and menu rebuild churn to keep startup and refreshes noticeably lighter.
- Fixed project grouping for newer Codex desktop state by honoring thread workspace-root hints, separating projectless chats, and handling `null` workspace labels.
- Eliminated duplicate project sections caused by worktree paths, Unicode-equivalent folder names, and stale project-catalog reloads.
- Fixed the notch menu overscroll surface so scrolling no longer reveals a mismatched background.
